# AI Voice Appointment Booking System

This project is an AI-powered automation system that handles customer calls and books appointments for salons and dental clinics.

## Features
- Automated call handling using AI voice agent
- Real-time appointment scheduling
- Calendar integration
- Backend automation using n8n / Make.com

## Workflow
Lead → Call → Availability Check → Booking → Confirmation
